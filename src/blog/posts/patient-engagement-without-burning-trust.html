Patient engagement without burning trust

There is a line between a helpful nudge and a spammy campaign, and it's thinner than most vendors admit.

Six rules keep us on the right side of it.

Rule one: never message a patient we do not have explicit consent from.

Rule two: never message a patient outside 9am–7pm local time, unless they've asked us to.

Rule three: never send more than one message per day per patient, unless the patient replies.

Rule four: every message must be actionable — confirm, reschedule, view invoice, tap for directions.

Rule five: every message must be in the patient's language.

Rule six: the opt-out must be one tap, and honoured immediately.

Break any of these and the message reads as spam, even when the content is helpful.

Honour all of them and the same content reads as care.

Engagement is a trust budget. Every message spends a little.

Every useful message earns a lot.

The math only works if you're miserly with the spends and generous with the earns.

Broadcasts are the biggest budget-burners. Use them sparingly, and only for content that the patient will thank you for.

1:1 messages, triggered by an actual event in the patient's care, are almost free. Send those often.

Reminders sit in the middle. They're only useful if the timing is right.

Feedback requests should come exactly once, immediately after the visit.

Recall campaigns should feel like a personal note from the doctor, not a marketing blast.

Every campaign we run against these rules performs worse. Every one that follows them performs better.

There's no cleverness here. Just discipline, applied every day.
